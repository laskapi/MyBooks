To run MyBooks app please create folder for your repo, 
get into and run following commands:

1. git clone https://github.com/laskapi/MyBooks
2. cd backend
3. docker compose up
4.  mvn spring-boot:run
5. cd ..
6. cd client
7. npm install
8. npm run dev

9. open your web browser and navigate to:
            http://localhost:5173/
